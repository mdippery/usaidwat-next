You will need access to the Claude API to run `usaidwat summary`.
To get access to the Claude API, create an API account at
[4mplatform.claude.com[24m. Once you have an account, save your API key in the
$CLAUDE_API_KEY environment variable and rerun `usaidwat summary`.
